Installation
============

yamu is currently developed as a Python package. Install it in a virtual
environment or via pip.

Using a virtual environment
---------------------------

::

    python -m venv .venv
    source .venv/bin/activate
    pip install -e .

Configuration
-------------

Create ``~/.config/yamu/config.yaml`` and add your Steam API key and ID. See
:doc:`/reference/config` for all available options.
